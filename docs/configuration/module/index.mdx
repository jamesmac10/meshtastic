Modules are included in the firmware and allow users to extend the functionality of their mesh or device.

|                                    Name                                    |                                                                                                             Description                                                                                                             |
| :------------------------------------------------------------------------: | :---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------: |
|       [Ambient Lighting](../configuration/module/ambient-lighting)     |                                                                                              Adjust the brightness of NCP5623 I2C RGB LEDs                                                                                          |
|                 [Audio](../configuration/module/audio)                 |                                                                                      Enable Support for Codec2 Voice Comms on certain devices.                                                                                      |
|        [Canned Message](../configuration/module/canned-message)        |                                                   Set a number of predefined messages to send out directly from the device with the use of an input device like a rotary encoder.                                                   |
|      [Detection Sensor](../configuration/module/detection-sensor)      |                                                                     Configure a GPIO pin to be monitored for specified high/low status and send text alerts.                                                                        |
| [External Notification](../configuration/module/external-notification) |                                                                Incoming messages are able to alert you using circuits you attach to the device (LEDs, Buzzers, etc).                                                                |
|                  [MQTT](../configuration/module/mqtt)                  |                                              Forward packets along to an MQTT server. This allows users on the local mesh to communicate with users on another mesh over the internet.                                              |
|          [Neighbor Info](../configuration/module/neighbor-info)        |                                                                                           Send info on 0-hop neighbors to the mesh.                                                                                                 |
|            [Range Test](../configuration/module/range-test)            | Send messages with GPS location at an interval to test the distance your devices can communicate. Requires (at least) one device set up as a sender and one as a receiver. The receiver(s) will log all incoming messages to a CSV. |
|       [Remote Hardware](../configuration/module/remote-hardware)       |                                                                                        Set and read a GPIO status remotely over the mesh.                                                                                           |
|            [Serial Module](../configuration/module/serial)             |                                                                                Send messages across the mesh by sending strings over a serial port.                                                                                 |
|  [Store & Forward](../configuration/module/store-and-forward-module)   |                                                                         Stores messages on a device for delivery after disconnected clients rejoin the mesh.                                                                        |
|             [Telemetry](../configuration/module/telemetry)             |                                                                        Attach sensors to the device and transmit readings on a regular interval to the mesh.                                                                        |
|            [Traceroute](../configuration/module/traceroute)            |                                                                                Track which nodes are used to hop a message to a certain destination.                                                                                |
